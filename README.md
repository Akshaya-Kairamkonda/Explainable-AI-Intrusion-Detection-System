Explainable AI for Intrusion Detection Systems

• Detects network intrusions using Machine Learning and Deep Learning algorithms.

• Uses Explainable AI techniques such as SHAP and LIME to interpret model predictions.

• Implemented using Python, Jupyter Notebook, Scikit-Learn, TensorFlow, and XGBoost.

• Algorithms Used:
  - MLP
  - LSTM
  - TCN
  - XGBoost
  - Voting Classifier

• Dataset:
  - CIC Intrusion Detection Dataset

• Key Features:
  - Intrusion Detection
  - Attack Classification
  - Feature Importance Analysis
  - SHAP Explanations
  - LIME Explanations
  - Performance Comparison

• Results:
  - MLP Accuracy: 99.22%
  - TCN Accuracy: 99.33%
  - LSTM Accuracy: 99.44%
  - XGBoost Accuracy: 99.77%
  - Voting Classifier Accuracy: 100%

• Developed as a B.Tech Final Year Project.
